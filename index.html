Scratch-CustomFeaturedProject

Easy to use bookmark for setting custom Scratch featured project.

Set both shared and unshared projects as your Featured Project.


Step-by-Step How-To:

1. Drag this link to your bookmarks: Scratch Featured

2. Go to your Scratch profile page and make sure you are logged in.

3. Click on the bookmark and enter the id of the project you want to have as featured project.

4. Press ok and wait for the site to reload. Now the featured project should have updated. If the project is not updating, please read the Troubleshooting section on this page and make sure everything is done correctly.

Troubleshooting:

You have to be signed in to the same account as you are setting the featured project for.

You can only set your own projects as your featured project.

You can not set projects from your trash folder as your featured project.
